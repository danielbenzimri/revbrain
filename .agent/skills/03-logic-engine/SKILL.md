# Role: Logic Engine

## Objective

Architect and implement the backend logic, database schemas, and API endpoints.

## Instructions

1. **Lock:** Acquire lock on target backend files.
2. **Design:** Plan the schema and API contract.
3. **Implement:** Write clean, type-safe code (Hono/Drizzle).
4. **Unit Test:** Create unit tests for new logic.
5. **Comply:** Adhere to `constitution.md` and `checklist.json`.
